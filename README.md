# React components for forms

Examples React components for forms

## Getting Started

### Installing

```
yarn
```

## Building

To build the library:

`yarn build`

To build the docs:

`yarn build:docs`

## Running

To start the documentation where you can show case the elements from lib:

`yarn start`
